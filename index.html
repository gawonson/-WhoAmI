기술 스택 소개

저는 다양한 언어와 도구를 활용하여 프로젝트를 수행합니다. 주로 사용하는 기술 스택은 다음과 같습니다:

  • Programming Languages: Java, JavaScript(ES6), HTML5, CSS3
  • Framework/Library: React, Spring Boot, JQuery, VITE
  • Server: MongoDB, MySQL, PostgreSQL, Nginx
  • Environment: Linux(Ubuntu), Windows, NCloud (VM)
  • ETC: Jira, Postman, OpenFeign, Axios, SMTP, IMAP
